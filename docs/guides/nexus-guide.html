NexusEngine

Motor personalizado con comandos unificados

JavaScript Personalizado Unificado

Configuración Inicial

Configuración Básica
// Inicializar NexusEngine
const nexus = new NexusEngine({
  canvas: 'gameCanvas',
  width: 800,
  height: 600,
  engine: 'auto' // Detecta automáticamente
});

// Iniciar el motor
nexus.init();
Loop Principal
// Función de actualización
nexus.onUpdate((deltaTime) => {
  // Lógica del juego
  player.update(deltaTime);
});

// Iniciar loop
nexus.start();

Comandos Unificados

Crear Objetos
// Comando universal para crear objetos
const player = nexus.create('sprite', {
  texture: 'player.png',
  x: 100,
  y: 100
});

// Funciona con cualquier motor
const box = nexus.create('box', {
  width: 50,
  height: 50
});
Transformaciones
// Comandos unificados de transformación
nexus.move(player, 200, 150);
nexus.rotate(player, 45);
nexus.scale(player, 1.5);

// Animaciones suaves
nexus.animate(player, {
  x: 300,
  duration: 1000
});
Controles
// Sistema de input unificado
nexus.input.onKey('SPACE', () => {
  player.jump();
});

nexus.input.onClick((x, y) => {
  nexus.move(player, x, y);
});

// Soporte para móvil
nexus.input.onTouch(handleTouch);

Características Especiales

Adaptación Automática
// NexusEngine se adapta automáticamente
// al motor gráfico disponible

// Si detecta Kaplay:
nexus.setEngine('kaplay');

// Si detecta Three.js:
nexus.setEngine('threejs');

// Los comandos siguen siendo los mismos
nexus.create('sprite', options);
Intérprete IA
// Comandos en lenguaje natural
nexus.ai.interpret("Crear un jugador en el centro");
nexus.ai.interpret("Mover el jugador a la derecha");
nexus.ai.interpret("Agregar física al jugador");

// El IA traduce a comandos del motor
nexus.ai.setLanguage('es'); // Español
nexus.ai.setLanguage('en'); // Inglés

Comandos Avanzados

Física Unificada
// Sistema de física universal
nexus.physics.enable();

// Agregar física a objetos
nexus.physics.add(player, {
  type: 'dynamic',
  bounce: 0.8
});

// Detectar colisiones
nexus.physics.onCollision(player, enemy, handleCollision);
Audio Universal
// Sistema de audio unificado
nexus.audio.load('jump', 'sounds/jump.wav');
nexus.audio.load('bgMusic', 'music/bg.mp3');

// Reproducir sonidos
nexus.audio.play('jump');
nexus.audio.playMusic('bgMusic', { loop: true });

// Control de volumen
nexus.audio.setVolume(0.8);

Atajos Útiles

Atajos de Edición
Ctrl + Z - Deshacer última acción
Ctrl + Y - Rehacer acción deshecha
Ctrl + S - Guardar proyecto
Ctrl + C - Copiar objeto seleccionado
Ctrl + V - Pegar objeto copiado
Ctrl + D - Duplicar objeto seleccionado
Delete - Eliminar objeto seleccionado
Control de Juego
Spacebar - Pausar/Reproducir juego
F5 - Reiniciar juego
F6 - Ejecutar paso a paso
F11 - Pantalla completa
Esc - Salir de pantalla completa
Tab - Cambiar entre modos de vista
Ctrl + R - Recargar escena actual
Nota: Estos atajos están disponibles cuando el motor NexusEngine está activo. Algunos atajos pueden variar según el motor gráfico subyacente utilizado.
